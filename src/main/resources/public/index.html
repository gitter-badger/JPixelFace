Information

PixelFace is a forum avatar service, like Gravatar but for Minecraft.

For news and announcements follow @TehRainbowGuy on Twitter.

All endpoints support user names, Mojang formatted UUIDs and properly formatted UUIDs.

New! All image styles support svg. https://PixelFace.net/helm/[name].svg

Avatars

Simply use https://PixelFace.net/avatar/[Name]/[Size]

Helms

Simply use https://PixelFace.net/helm/[Name]/[Size]

Torsos

Simply use https://PixelFace.net/torso/[Name]/[Scale]

Please note that the size for this is x = 18 * scale, y = 20 * scale

Busts

Simply use https://PixelFace.net/bust/[Name]/[Size]

Full body

Simply use https://PixelFace.net/body/[Name]/[Scale]

Please note that the size for this is x = 18 * scale, y = 32 * scale

Skins

Simply use https://PixelFace.net/skin/[Name]

Profiles

Simply use https://PixelFace.net/profile/[Name].json
